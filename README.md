# StudySquad Backend API 

**StudySquad** is a Django-based backend for a student study group matchmaking platform. It connects students who share the same courses and have overlapping free time.

## Project Status
* **Week 1:** Authentication & User Identity (Completed)
* **Week 2:** Academic Data & Scheduling Engine (Completed)
* **Week 3:** Matchmaking Algorithm & Groups (Upcoming in the coming week )
